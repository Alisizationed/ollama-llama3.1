<h1>Ollama</h1>

<p>It is a Java project for sending generation requests to ollama, to llama3.1 image.</p>
<p>It uses such technologies:</p>
<ul>
  <li>Spring Boot Webflux</li>
  <li>Spring Boot Security</li>
  <li>Swagger</li>
  <li>Ollama</li>
  <li>Docker</li>
  <li>Keycloak</li>
</ul>
